  • Ingrid is a small Sass based grid system that solves big problems. It's responsive, fluid, or a mix of both, and it's incredibly easy to customise and use.
  • Ivy is a work-in-progress icon set for displays of all resolutions. It aims to successfully convey messages whether viewed on retina displays or CRT.
  • Ella is a growing collection of modular, reusable illustrations intended for use in the construction of user interfaces.
  • Design Digest is a curated reading list for curious Designers, and just about anyone else. The tone of the articles featured is less technical, and more personal.
  • Palette is a simple page for browsing a large variety of colors for use on the web. It's a Sass loop that can be used locally to generate different colours on save.
  • Shape & Color is a tool for considering how the colors we choose act and interact on the web, in the hope that we can begin to pay them more consideration.
  • Flow is a set of Sketch symbols that can be used to quickly and easily mock up a user flow with tiles that represent the content the states contain.
